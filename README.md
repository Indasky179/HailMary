# HailMary
 This project implements a smart wireless surveillance system using the ESP32-CAM module. It allows real-time video streaming over WiFi, captures images, and records frames to an SD card. A push button is integrated to trigger photo capture and toggle recording, making it suitable for portable security applications.  The system is powered using a LiPo battery with a charging module, making it fully wireless and compact.   ---  ⚙️ Features:  📸 Capture photo via browser or button  🎥 Live video streaming over WiFi  💾 Frame-based recording to SD card  🔘 Push button control (photo + recording toggle)  📡 Wireless access via IP address  🔋 Battery-powered portable system    ---  🧰 Hardware Used:  ESP32-CAM (AI Thinker)  LiPo Battery  TP4056 Charging Module  Push Button  Jumper Wires    ---  🧪 Working:  ESP32 connects to WiFi  Streams video via web server  Button press:  Captures image  Toggles recording   Frames stored in SD card    ---  🌐 Access:  After connecting to WiFi, open in browser:  http://<ESP32-IP>/stream   → Live video   http://<ESP32-IP>/capture  → Capture photo   http://<ESP32-IP>/start    → Start recording   http://<ESP32-IP>/stop     → Stop recording   ---  ⚠️ Note:  Video recording is frame-based (not real MP4 due to hardware limitations)  ESP32 supports only 2.4GHz WiFi    ---  🚀 Future Improvements:  Motion detection  Mobile app integration  Cloud storage  Face recognition
